# QTCI Complete

This folder contains a more advanced and exploratory notebook centered on **QTCI / TCI-style workflows**, implementation details, and tensor-based approximations of structured operators.

## Contents

- `QTCI_completo.ipynb`

## Overview

This notebook gathers a broader set of experiments related to **quantics-based representations** and **tensor cross interpolation (TCI)**. Compared with the more compact and presentation-oriented notebooks in the repository, this one is more technical and exploratory.

It is useful for understanding the computational side of the project in more detail, including:

- how structured operators are queried and encoded,
- how interpolation-based tensor methods are implemented,
- how pivots, ranks, tolerances, and encodings affect the results,
- and how tensor approximations behave in practice for the problems studied here.

## Main topics

The notebook may include topics such as:

- quantics representations of operators,
- tensor cross interpolation (TCI / QTCI),
- pivot selection and sampling strategies,
- comparison between exact and approximate representations,
- rank and bond-dimension behavior,
- reconstruction tests,
- and more technical numerical experiments.

## Why this notebook matters

This is one of the most technically rich notebooks in the repository.  
It reflects the more exploratory side of the work: testing ideas, implementations, and tensor-interpolation workflows beyond the most polished examples.

For readers interested in the methodological and algorithmic side of the project, this notebook provides a more detailed view of how the tensor-based approximations were actually explored.

## Role within the repository

This notebook should be seen as an **advanced companion notebook**.  
It is not necessarily the best starting point for a first reading, but it is very valuable for understanding the deeper numerical and implementation aspects of the project.
